PairNet: Object Detection and segmentation Tool

A system for the detection and segmentation of target features especially in the medical images. PairNet is a deep learning pipeline that utilizes the proposed Reinforced YOLOv4-tiny and Mask R-CNN-lite deep-learning based architectures that are optimized for low powered devices such as CPU dependant devices, IoT devices etc. thus improving the accessibility of the system.
